# Mathpad
A 4x4 Macro-pad with OLED Display and per key addressable LEDs. 

Allows for effortless and straightforward math symbol input.


Seeed XIAO RP2040
16x MX Switches
16x Keycaps
16x THT  1N4148 Diodes
0.91 inch OLED display (128x32)
16x SK6812 MINI-E LEDs
4x M3x16mm screws
4x heatset inserts

3D Printer
Soldering Iron
